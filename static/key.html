Key Loggers

Key loggers are programs designed to log all the key strokes typed by a computer user. They can record all the things typed into the keyboard and send them to an nefarious user or store them for later retrieval. Once the data is captured, it must be analysed to determine if there is any useful content.

Goal: Steal the usernames and passwords from the computer user.
